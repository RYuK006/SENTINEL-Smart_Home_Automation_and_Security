# Monsoon and Dengue Defense

As dusk approaches, if [Sentinel Grill](../Core-Components/Sentinel%20Grill.md) confirms windows are open, [Sentinel Swarm](../Core-Components/Sentinel%20Swarm.md) automatically activates to repel mosquitoes. If a flood sensor detects rising water in the basement, [Sentinel Valve](../Core-Components/Sentinel%20Valve.md) shuts off the main water line to prevent backflow or pipe bursts.
