# Nursery Threat Lockdown

If an intrusion is detected elsewhere in the house, or if [Sentinel Cradle](../Core-Components/Sentinel%20Cradle.md) detects a sudden anomaly (like a window breaking), the system instantly locks the nursery door, engages [Sentinel Shield](../Core-Components/Sentinel%20Shield.md) on the windows, and prioritizes parents' alerts.
