# Sentinel Tag

Small, coin-sized BLE trackers that you attach to high-value items (laptops, wallets, TVs). If the system is "Armed" and a Sentinel Tag is detected leaving the geofence, the system immediately flags a theft-in-progress and logs the item as stolen evidence.
