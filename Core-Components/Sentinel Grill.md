# Sentinel Grill

A specialized vibration, tension, and state sensor designed for the iron window grills common in Indian homes. If someone tries to pry or cut the grill, it triggers a high-severity alert before they even touch the window glass. It also tracks if the grill/window is left open, directly controlling [Sentinel Swarm](Sentinel%20Swarm.md).
