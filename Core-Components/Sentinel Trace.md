# Sentinel Trace

A covert GPS/BLE tracker designed to be permanently concealed inside high-value vehicles (cars, motorcycles, bicycles). It remains dormant to save battery, but if the vehicle moves while the system is armed, it instantly 'wakes up', triggers a theft alert, and provides live GPS tracking through the Sentinel network.
