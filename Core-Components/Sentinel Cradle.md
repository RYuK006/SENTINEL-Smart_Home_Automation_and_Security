# Sentinel Cradle

An advanced infant and nursery hub utilizing edge-AI. Beyond standard video, it tracks a baby's breathing rate via micro-radar (no wearables needed) and classifies audio to distinguish between types of cries. In a home-intrusion scenario, it integrates with the main hub to automatically lock the nursery door via [Selective Smart-Lock Logic](../Features/Selective%20Smart-Lock%20Logic.md).
